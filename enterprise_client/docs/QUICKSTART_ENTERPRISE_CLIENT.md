# 企业独立客户端快速启动

## 1. 启动
- 双击 `start_enterprise_client.bat`
- 等待浏览器自动打开 `http://127.0.0.1:8010/`
- 如果需要在演示前先做接口预检，可运行：

```bat
backend\venv\Scripts\python.exe backend\scripts\enterprise_client_login_smoke.py
```

## 2. 登录
- 使用企业演示账号或企业真实账号登录
- 支持的角色：
  - 企业管理员
  - 企业操作工
  - 企业后勤岗

## 3. 数据说明
- 如果你原本已经在主程序里保存了地图、地图模板、任务模板，企业客户端会优先尝试复用原主程序数据
- 若要强制指定数据源，可在运行目录放置 `.env`，或显式设置：
  - `AGV_DATA_BACKEND`
  - `AGV_DATABASE_URL`

## 4. 演示建议
- 企业管理员：
  - 查看企业概览、审计、AI 素材和拓扑配置
- 企业操作工：
  - 关注任务执行、主干道运行和 AGV 调度
- 企业后勤岗：
  - 关注地图方案、点位模板、运行设置与反馈

## 5. 示例资源
- `demo/docs/`：导师演示步骤与封装说明
- `demo/json/`：企业演示任务与地图示例

## 6. 常见说明
- 若端口 `8010` 被占用，请先关闭旧实例再重试
- 若希望企业客户端完全独立运行，可单独准备一套数据库配置，不与主程序共享
