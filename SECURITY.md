# Security Policy

## Supported versions

安全修复优先应用于当前 `3.x` 产品化分支。`v2.0.0` 是毕业设计稳定基线，不承诺持续安全维护。

## Reporting a vulnerability

请不要在公开 Issue 中披露可利用的漏洞、凭据或真实数据。请通过 GitHub 的私密漏洞报告功能联系维护者，并提供：

- 受影响版本和部署方式；
- 最小复现步骤；
- 可能影响；
- 建议修复方式（如有）。

维护者会先确认收到报告，再评估严重性、修复范围和披露时间。

## Deployment boundary

本项目当前是教学、研究和企业试用软件，不是经过工业安全认证的 AGV 控制器。真实设备接入、功能安全、紧急制动和现场风险控制不在 `v3.0.0` 的承诺范围内。

演示账号只能用于本机演示。企业试用环境必须使用持久化数据库、独立管理员、受控来源、HTTPS 和安全 Cookie，并关闭演示账号初始化。
