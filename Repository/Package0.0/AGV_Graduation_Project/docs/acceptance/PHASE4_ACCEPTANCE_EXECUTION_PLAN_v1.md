## 4. 口头讨论过但尚未完全收口的内容

### 4.1 聚合站点统一语义

已确认采用以下方向：

- 聚合后的停车站、充电站，在**运行态主地图/小地图**统一按“一个完整站点模块”显示
- 主地图只保留：

  - `P 1/4`
  - `P 0/4`
  - 多个内部点位/泊位的可见概念
- 内部泊位与站内 AGV 信息下沉到详情层

### 4.2 编辑态与运行态分离

- 运行态主地图：按站级模块展示
- 企业拓扑编辑器：继续保留点位级编辑语义
- 本轮不把编辑态强行改造成单一大站点按钮

### 4.3 常用点位与拓扑节点语义分离

- 常用点位只表示“任务起终点业务地标”，例如入库口、出库口、存储位、装配位、质检台
- 常用点位不再默认提供“充电区 / 充电位”，避免和路网拓扑里的充电点、充电站含义重叠
- 回仓、回充、站内容量、充电状态只以路网拓扑里的停车点、充电点和聚合站点为准
- 主地图和小地图会显示常用点位的独立业务标记，但它们不参与调度容量和自动回充逻辑
