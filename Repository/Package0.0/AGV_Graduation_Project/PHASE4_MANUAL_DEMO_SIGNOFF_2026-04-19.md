## 4. 必看演示项

### 4.1 路网拓扑与主干道

- 打开企业设置里的路网拓扑入口
- 修改一个安全的小拓扑配置并保存
- 关闭后重新打开，确认配置没有丢失
- 派发一条能接入主干道的任务，观察普通段、主干道、离开主干道的路线表现

通过标准：

- 拓扑入口可打开
- 保存后配置可恢复
- 路径不绕远到不合理的主干道终点
- AGV 在主干道上速度表现符合倍率配置

### 4.2 停车站 / 充电站

- 查看横向、竖向、L 形或异形聚合站点
- 确认主地图只显示站级 `P/C + n/m`
- 悬停站点，确认能看到站内 AGV 编号或空站提示
- 点击站点，确认可以打开站内详情
- 让多台 AGV 回仓或回充，观察站内容量统计

通过标准：

- 站点不像多个碎片点位
- 站级 `n/m` 不被 AGV 图标遮挡
- 第二台及后续车辆不会在容量允许时长期堵在站外
- 站内车辆列表与主地图占用数一致

### 4.3 说明中心与设置弹窗

- 首次进入企业端，确认说明中心自动打开
- 按 `H` 重新打开说明中心
- 打开企业设置或页面设置后，再打开说明中心
- 按 `Escape` 关闭说明中心

通过标准：

- 说明中心永远在设置弹窗之上
- `H` 和 `Escape` 行为稳定
- 说明中心里能读到主干道、等待让行、回仓回充等说明

### 4.4 通知与反馈铃

- 创建或查看企业内部请求
- 查看平台 Bug 反馈入口
- 观察右下角小铃铛与返回顶部按钮的位置关系

通过标准：

- 小铃铛未读数字可见
- 未读 / 待处理状态有视觉变化
- 小铃铛不遮挡返回顶部按钮

### 4.5 长时间运行观感

- 派发至少 2 台 AGV 经过交汇口或共享走廊
- 观察 3 到 5 分钟
- 记录是否出现等待、让行或恢复

通过标准：

- 不同 AGV 不同格重叠
- 后车不穿过前车
- 等待 / 让行后能恢复，不长期卡死
- 任务完成后 `current_edge`、路线、起终点显示能正常收口

### 4.6 常用点位语义

- 打开常用点位面板，确认预设点位用于任务起终点快捷填充
- 查看主地图和小地图，确认常用点位有独立业务标记
- 确认默认常用点位不再出现“充电区 / 充电位”
- 确认真正的回充能力仍来自路网拓扑里的“充电点 / 充电站”

通过标准：

- 常用点位和拓扑充电点不会在文案上混为一类
- 点击“定位地图”能聚焦到对应业务点位
- 站点容量 `n/m` 仍只来自停车点、充电点或聚合站点
