# 导师演示步骤 v2

## 一、启动
1. 双击 `start_agv.bat`
2. 等待浏览器自动打开前端页面
3. 保持默认 SQLite 数据即可开始演示

## 二、先演示平台管理员治理工作台
1. 使用平台管理员演示账号登录
2. 默认展示：
   - 企业审批
   - 账号治理
   - 审计摘要
3. 打开账号治理，演示：
   - 新注册个人用户
   - 企业账号
   - 封禁 / 解封 / 停用
   - 导出 JSON / CSV

## 三、再演示平台管理员测试入口
1. 点击 `测试个人界面`
2. 说明这是平台管理员的排错预览入口，不改变真实个人用户界面
3. 返回治理工作台
4. 点击 `测试企业界面`
5. 切换企业管理员 / 企业操作工 / 企业后勤岗预览

## 四、导入示例 JSON
### 任务示例
- `task_auto_single_demo.json`
- `task_manual_single_demo.json`
- `task_multi_stage_demo.json`

### 地图示例
- `obstacle_layout_demo.json`
- `map_profile_irregular_demo.json`

## 五、企业能力演示建议
1. 打开企业设置
2. 导入异形地图方案
3. 使用网格编辑器补充 / 删除有效格
4. 导入任务示例
5. 运行单段与多段任务
6. 演示 AI 素材与审计记录

## 六、收尾说明
- 当前封装版重点是“治理工作台优先 + 企业版升级预览”
- 个人用户界面保持既有显示，不纳入第四阶段重构范围
