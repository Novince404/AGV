# AGV Dispatch Package v2 Demo Assets

这个目录会随着 `AGV_Dispatch_Package_v2` 一起封装，方便导师演示时开箱即用。

## 目录说明
- `docs/`
  - 演示步骤说明
  - SQLite 快速启动说明
  - 最小交付版演示脚本
  - 故障排查说明
- `json/`
  - 自动单段任务示例
  - 手动单段任务示例
  - 多段任务示例
  - 障碍布局示例
  - 异形地图方案示例

## 推荐演示顺序
1. 运行 `start_agv.bat`
2. 使用内置账号进入系统
3. 先演示平台管理员治理工作台
4. 再导入 `demo/json/` 中的任务与地图示例
5. 进入企业界面，演示地图、任务、AI 素材等能力

## 说明
- 演示包默认使用内置 SQLite 数据库，因此首次启动不需要额外准备数据库。
- JSON 示例不是唯一数据来源，而是用于快速恢复导师演示场景的最小素材集。
