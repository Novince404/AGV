# AGV 企业独立客户端

这个封装包用于企业端独立演示与企业内预览。

## 适用身份
- 企业管理员
- 企业操作工
- 企业后勤岗

## 默认行为
- 启动后直接进入企业独立客户端登录页
- 不再提供游客、个人用户、平台管理员入口
- 默认监听地址：`http://127.0.0.1:8010/`

## 数据源优先级
企业独立客户端现在会**优先复用原有主程序数据**，避免打开后像换了一套地图与模板。

优先级如下：
1. 如果已显式设置 `AGV_DATA_BACKEND` / `AGV_DATABASE_URL`，则直接使用外部配置
2. 如果运行目录存在 `.env`，则读取该配置
3. 如果检测到原主程序数据库，则优先复用：
   - 仓库根目录 `data/agv_dispatch.db`
   - 同级封装包 `AGV_Dispatch_Package_v2/data/agv_dispatch.db`
   - 同级封装包 `AGV_Dispatch_Package/data/agv_dispatch.db`
4. 如果以上都不存在，再回退到企业客户端本地数据库

## 封装内资源
- `docs/`：企业客户端说明与快速启动文档
- `demo/docs/`：导师演示说明与步骤
- `demo/json/`：企业演示任务与地图示例 JSON

## 说明
- 企业独立客户端仍是企业端专用入口，但默认会尽量复用你原本已经调好的地图、模板和任务数据
- 如果需要完全独立的数据环境，可以单独提供 `.env` 或设置 `AGV_DATABASE_URL`
