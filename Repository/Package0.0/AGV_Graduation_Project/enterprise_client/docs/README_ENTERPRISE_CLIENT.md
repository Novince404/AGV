# AGV 企业独立客户端

这个封装包用于企业侧独立演示和企业内部部署预览。

## 适用身份
- 企业管理员
- 企业操作工
- 企业后勤岗

## 默认行为
- 启动后直接进入企业独立客户端登录界面
- 不再提供游客、个人用户、平台管理员入口
- 默认使用本地 SQLite 数据库：`data/agv_enterprise_client.db`
- 默认监听地址：`http://127.0.0.1:8010/`

## 内置资源
- `docs/`：企业客户端说明与快速启动文档
- `demo/json/`：企业演示任务与地图示例 JSON

## 说明
- 本封装包是企业独立客户端阶段交付物，不等同于通用平台封装包。
- 若本地没有现成数据库，首次启动会自动创建。
