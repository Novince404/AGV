# 企业独立客户端快速启动

## 1. 启动
- 双击 `start_enterprise_client.bat`
- 等待浏览器自动打开 `http://127.0.0.1:8010/`

## 2. 登录
- 使用企业演示账号或企业真实账号登录
- 支持的演示角色：
  - 企业管理员
  - 企业操作工
  - 企业后勤岗

## 3. 演示建议
- 企业管理员：
  - 查看企业概览、审计、AI 素材
- 企业操作工：
  - 关注调度执行与任务链路
- 企业后勤岗：
  - 关注地图方案、点位模板与运行配置

## 4. 示例资源
- `demo/json/` 提供企业演示任务与地图示例
- 若任务导入失败，请先确认当前地图与示例 JSON 匹配

## 5. 常见说明
- 数据库文件位于 `data/agv_enterprise_client.db`
- 若端口 `8010` 被占用，请先关闭旧实例后重试
