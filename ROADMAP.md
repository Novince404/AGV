# AGV Roadmap

## v3.0.0-beta.2：工程基线

- 仓库根目录整理、统一版本和持续集成。
- `/api/v1`、健康检查、统一错误和兼容层。
- 自动化测试入口、社区文档和发布检查。

## v3.0.0-beta.3：数据与身份

- Alembic、备份、升级和校验。
- Argon2id、本地安全会话、CSRF 和登录限流。
- 通用 OIDC 与 Keycloak 验收环境。

## v3.0.0-beta.4：调度运行时与新工作台

- API 与 scheduler 分离、确定性时间步、事件和设备适配层。
- 路由化全角色工作台、统一 API Client 和实时事件时间线。

## v3.0.0-rc.1

- Windows 包、Docker Compose、MySQL、监控、升级和恢复演练。
- 完整浏览器回归、迁移测试和长时间仿真。

## v3.0.0

- 无未解决的 P0/P1 缺陷。
- RC 连续观察至少 7 天。
- 发布包、校验值、OpenAPI、升级文档和已知限制齐全。

## 后续方向

- MQTT、PLC、串口等真实设备适配器。
- 传感器级局部避障和连续速度规划。
- 高可用调度、站级实体和更大规模基准实验。
